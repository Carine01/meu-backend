# Contribuindo para meu-backend

Obrigado por considerar contribuir!

## Como contribuir
- Faça um fork do projeto
- Crie uma branch para sua feature/fix: `git checkout -b minha-feature`
- Commit suas mudanças: `git commit -m 'feat: minha nova feature'`
- Push para sua branch: `git push origin minha-feature`
- Abra um Pull Request

## Convenções
- Escreva commits claros e descritivos
- Sempre adicione testes para novas funcionalidades
- Siga o padrão de código do projeto
